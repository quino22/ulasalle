  • Descripción:
    • En este ejemplo se muestran los tipos de relaciones entre objetos (uno-a-uno y uno-a-muchos).
    • Una persona puede tener un color. (Persona02-Color)
    • Una persona puede tener varios colores. (Persona03-Color)
  • Arquitectura y Plataforma de desarrollo:
    • Linux dell 3.19.0-16-generic #16-Ubuntu SMP Thu Apr 30 16:13:00 UTC 2015 i686 i686 i686 GNU/Linux
    • Eclipse Java EE IDE for Web Developers. Version: Luna Service Release 2 (4.4.2) Build id: 20150219-0600
    • Google Plugin para Eclipse Luna
    • App Engine Java SDK 1.9.19
  • Para que este ejemplo pueda publicarse en el hosting de Google App Engine, usando Deploy to App Engine..., se recomienda:
    • Utilizar como la versión 1.7 del compilador java (La versión 1.8 no esta soportada hasta este momento)
    • Edite el archivo eclipse/eclipse.ini y agregar antes de -product, la ruta de la máquina virtual (vm) java. Ejemplo:
      -vm
      /usr/lib/jvm/java-1.7.0-openjdk-i386/bin/java
    • Reiniciar Eclipse y despues utilice Deploy to App Engine...
